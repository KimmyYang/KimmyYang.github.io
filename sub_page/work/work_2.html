interface裡面的class defualt 都是abstract, 所以裡面的method都須備實作
abstract 裡面可以有concrete methods(一般自訂method), 或是abstract method, 繼承abstract 的class需實作定義為abstract的method

interface 只能extends interface
abstract 可以extends abstract 和 一般的class

interface A {
    //Interface can have only static final (constant) variable i.e. by default
    int aa = 10;
    public void a1();
    public void a2();
}

class test extends A{
    public void a1(){ System.out.println("a1");}
    public void a2(){ System.out.println("a2");}
}

abstract B{
    //Abstract class can have static, final or static final variables with any access specifier
    private int bb = 5;
    protected final int bbb = 10;
    public void b1(){ System.out.println("b1");}
    public  abstract void b2();
    protected abstract void b3();
}

class test_2 extends B{
    public void b2(){ System.out.println("b2");}
    protected void b3() {System.out.println("b3");}
}
